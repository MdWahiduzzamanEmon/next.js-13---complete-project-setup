## Getting Started

# This template contains the following features:

- [x] Next.js
- [x] TypeScript
- [x] Tailwind CSS
- [x] ESLint
- [x] Prettier
- [x] Formatter
- [x] PWA(Progressive Web App)
- [x] SEO
- [x] Sitemap
- [x] Robots.txt

## First, install the dependencies by running the following command in the project directory.

```bash
npm install
# or
yarn
# or
pnpm install
```

## Then, run the development server to open the project in port 3000.

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
```

## To open dev server in different port

```bash

start-next
# or
npm run dev -- -p <port>
# or
yarn dev -- -p <port>
# or
pnpm dev -- -p <port>
```

## To build the project

```bash
npm run build
# or
yarn build
# or
pnpm build
```

## To start the project in production mode

```bash
npm run start
# or
yarn start
# or
pnpm start
```
